◆ check_composto()

vector_operations.check_composto ( number)

Verifica se um número é composto.

A função considera que apenas números maiores ou iguais a 2 podem ser compostos. Em seguida, testa se o número possui algum divisor além de 1 e dele próprio. Caso exista um divisor, o número é classificado como composto; caso contrário, é primo.

Parameters
numberNúmero inteiro a ser analisado.
Returns
True se o número for composto; False caso contrário.

◆ get_numbers_from_user()

vector_operations.get_numbers_from_user ( mensagem_inicial = None)

Lê um conjunto de números inteiros inseridos pelo utilizador.

Esta função solicita ao utilizador que introduza um número fixo de valores inteiros (entre -10 e 27). Cada valor é validado individualmente; se estiver fora do intervalo ou não for inteiro, é pedido novamente. Caso seja fornecida uma mensagem inicial, esta é exibida antes da leitura dos números, permitindo diferenciar a leitura do vetor inicial da leitura de um novo vetor na opção 8.

Parameters
mensagem_inicialMensagem opcional apresentada antes da leitura dos números (utilizada, por exemplo, na opção 8).
Returns
Lista contendo todos os valores válidos introduzidos pelo utilizador.

◆ init_program()

vector_operations.init_program ( )

Inicia o programa e gere o ciclo principal do menu.

Esta função trata do caso especial em que o programa é executado com a flag '–help', mostrando a página de ajuda. Caso contrário, lê inicialmente um vetor de números inteiros fornecido pelo utilizador e entra num ciclo em que apresenta o menu principal, lê a opção escolhida e chama a função correspondente. As operações incluem cálculos sobre o vetor inicial, criação de matrizes 2x10 e 10x10, geração da transposta e leitura de um segundo vetor para a soma com o dobro dos seus valores. O ciclo termina quando o utilizador seleciona a opção 0.

Returns
Não devolve valores; coordena apenas a execução das restantes funções.

Gera uma matriz 10x10 com o produto entre cada valor do vetor e o vetor ordenado.

A função ordena o vetor recebido por ordem crescente e, para cada elemento do vetor original, calcula o produto com todos os elementos do vetor ordenado. O resultado é organizado numa matriz 10x10, onde cada linha corresponde a um valor original multiplicado por todos os valores ordenados.

Returns

◆ soma_primeiro_com_dobro_segundo()

vector_operations.soma_primeiro_com_dobro_segundo ( numbers)

Soma cada elemento do vetor inicial com o dobro dos elementos de um novo vetor.

A função solicita ao utilizador um segundo vetor com o mesmo tamanho através de get_numbers_from_user(). Em seguida, combina os dois vetores elemento a elemento, somando cada valor do vetor inicial com o dobro do valor correspondente do segundo vetor. A operação é feita utilizando a função zip() para emparelhar os elementos.

Parameters
numbersVetor inicial previamente introduzido pelo utilizador.
Returns
Um novo vetor resultante da operação: elemento_inicial + 2 × elemento_novo.

vector_operations.valor_mais_proximo ( numbers,
target = 15 )

Determina o valor do vetor que está mais próximo do alvo indicado.

A função percorre todos os elementos do vetor e calcula a diferença absoluta entre cada valor e o alvo (target). O elemento cuja diferença for menor é considerado o mais próximo. O valor por defeito do alvo é 15, mas pode ser alterado se necessário.

Parameters
numbersVetor de números inteiros onde será feita a pesquisa.
targetValor de referência para o qual se procura o elemento mais próximo.
Returns
O valor do vetor cuja distância ao alvo é mínima.
