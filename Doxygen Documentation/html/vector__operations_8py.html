Detailed Description

Programa interativo para processamento de vetores e matrizes.

Este programa permite ao utilizador introduzir um conjunto de valores inteiros e executar várias operações, incluindo cálculo de medidas, transformações do vetor, geração de matrizes e operações de produto e transposição. O menu apresentado ao utilizador organiza todas as funcionalidades disponíveis, incluindo uma opção de ajuda adicional.
