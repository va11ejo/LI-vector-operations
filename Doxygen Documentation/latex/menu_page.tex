\hypertarget{menu_page_autotoc_md18}{}\doxysection{\texorpdfstring{Página do Menu de Operações}{Página do Menu de Operações}}\label{menu_page_autotoc_md18}
Esta página descreve as opções do menu principal do programa {\bfseries{\doxylink{vector__operations_8py}{vector\+\_\+operations.\+py}}}. Cada opção corresponde a uma função específica, documentada no código-\/fonte, que trabalha sobre o vetor inicial ou sobre as matrizes geradas a partir desse vetor.

Após a introdução do vetor inicial, o menu é apresentado repetidamente até o utilizador escolher a opção {\ttfamily 0} (terminar o programa).

\DoxyHorRuler{0}
\hypertarget{menu_page_autotoc_md20}{}\doxysubsection{\texorpdfstring{Lista de Opções do Menu}{Lista de Opções do Menu}}\label{menu_page_autotoc_md20}

\begin{DoxyEnumerate}
\item {\bfseries{Determinação do elemento mais próximo de 15}} ~\newline
 Calcula qual o valor do vetor cuja distância a 15 é mínima.
\item {\bfseries{Cálculo do cosseno da segunda metade do vetor}} ~\newline
 Aplica a função cosseno aos elementos da segunda metade do vetor inicial.
\item {\bfseries{Reordenamento do vetor por ordem decrescente}} ~\newline
 Devolve o vetor inicial reordenado do maior para o menor valor.
\item {\bfseries{Extração dos valores que não são divisíveis por 3}} ~\newline
 Cria um novo vetor apenas com os elementos cujo resto da divisão por 3 é diferente de zero.
\item {\bfseries{Construção de uma matriz 2x10 com o quádruplo dos valores}} ~\newline
 Gera uma matriz 2x10 em que a primeira linha é o vetor inicial e a segunda linha contém os mesmos valores multiplicados por 4.
\item {\bfseries{Cálculo da média do vetor}} ~\newline
 Calcula e apresenta a média aritmética de todos os elementos do vetor.
\item {\bfseries{Ajuda adicional}} ~\newline
 Mostra uma página de ajuda com uma breve explicação do propósito do menu e um contacto fictício.
\item {\bfseries{Leitura de um novo vetor e soma com o dobro dos seus valores ao vetor inicial}} ~\newline
 Lê um segundo vetor com o mesmo tamanho e mostra a soma entre o vetor inicial e o dobro de cada elemento do novo vetor.
\item {\bfseries{Números compostos no vetor inicial}} ~\newline
 Identifica e apresenta todos os números compostos (não primos) que existem no vetor inicial.
\item {\bfseries{Matriz 10x10 do produto entre o vetor inicial e o mesmo vetor ordenado por ordem crescente}} ~\newline
 Gera uma matriz 10x10 em que cada linha corresponde a um elemento do vetor inicial multiplicado por todos os elementos do vetor ordenado.
\item {\bfseries{Matriz 10x10 transposta (da matriz de produtos)}} ~\newline
 Calcula e apresenta a matriz transposta da matriz produzida na opção 10.
\end{DoxyEnumerate}
\begin{DoxyEnumerate}
\item {\bfseries{Terminar o programa}} ~\newline
 Encerra a execução do programa. 
\end{DoxyEnumerate}
